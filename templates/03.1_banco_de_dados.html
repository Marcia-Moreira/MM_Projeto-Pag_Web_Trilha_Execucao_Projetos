No VsCode, editar comandos para a conexão com Banco de Dados através da Interface Gráfica DBeaver. Digite os comandos SQL no VS Code e atualize a página do DBeaver para ver as tabelas criadas

Codigo de Conexão DB x Python

- crie o arquivo de banco de dados .db na pasta de destino do projeto que deve ser a mesma pasta onde ficará o arquivo de codigo python!
- no DBeaver = arquivo/ Novo / Assistente-Pasta DBeaver / Conexão com Banco de Dados / Avançar / Todos / Escolha qual será seu Banco Utilizado (Neste Caso SQLite) / Avançar / Caminho ABRIR - procure o arquivo .db que vc criou na pasta do código Python. / Concluir. (deverá aparecer o arquivo .db na aba de navegação do DBeaver)
- Agora volte pro VScode para editar o código de conexão e os comandos de criação de tabelas e dados.
